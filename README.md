# Good CFA 

## Install

```
npm i
```

## Test

```
npx hardhat test
```

## Deploy

Create a `.env` on the model of `.env.example`:

```js
cp .env.example .env
```

Add your own keys in your `.env` file. 

Deploy to `alfajores` (Celo Testnet):

```
npx hardhat run scripts/clear.ts
npx hardhat run scripts/deploy.ts --network alfajores
```
